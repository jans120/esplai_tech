Ejercicio 05 - Entidades HTML

Caracteres especiales y sus entidades HTML

Punto y coma (;): &semi; o &#59; → ;

Menor que (<): &lt; o &#60; → <

Mayor que (>): &gt; o &#62; → >

Ampersand (&): &amp; o &#38; → &

Comilla simple ('): &apos; o &#39; → '

Comilla doble ("): &quot; o &#34; → "

Backslash (\): &#92; → \

Barra (/): &#47; → /

Copyright (©): &copy; o &#169; → ©

Espacio sin salto: &nbsp; o &#160; → palabra unida

Euro (€): &euro; o &#8364; → €
